# 🚀 启动 Frame Zero 前端项目

## 方式一：使用开发脚本（推荐）

```bash
./dev-start.sh
```

## 方式二：手动启动前端

### 1. 安装依赖

```bash
cd frontend
npm install
```

**注意**: 首次安装可能需要 2-5 分钟，取决于网络速度。

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 访问应用

打开浏览器访问: **http://localhost:3000**

5. **VSCode Dark+主题** - 完全复刻VSCode的深色主题

## 🔧 故障排查

### 端口被占用

如果 3000 端口被占用：

```bash
# 查看占用进程
lsof -i :3000

# 终止进程
kill -9 <PID>
```

### 依赖安装失败

清理并重新安装：

```bash
cd frontend
rm -rf node_modules package-lock.json
npm install
```
